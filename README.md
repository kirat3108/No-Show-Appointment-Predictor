# AI-Based No-Show Appointment Predictor

## Overview

The AI-Based No-Show Appointment Predictor is a Machine Learning project developed to predict whether a patient is likely to miss a scheduled hospital appointment. The system helps hospitals reduce appointment wastage, improve scheduling efficiency, and identify high-risk patients in advance.

## Features

* Predicts patient no-show probability
* Interactive web application using Streamlit (https://streamlit.io?utm_source=chatgpt.com)
* Risk classification (Low, Medium, High)
* Data visualization and analysis
* Machine Learning-based prediction system
* User-friendly interface

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn
* Pickle
* Streamlit

## Machine Learning Algorithm

* Random Forest Classifier

## Dataset

Dataset used: Kaggle Medical Appointment No-Show Dataset

Main features used:

* Age
* Gender
* Scholarship
* Hypertension
* Diabetes
* Alcoholism
* Handicap
* SMS Received
* Waiting Days
* Appointment Day
